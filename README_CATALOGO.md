## 📁 Como Usar

### Opção 1: Sua imagem se chama "catalogo.jpg"
```
1. Coloque o arquivo em: TenDigitalMZ_v3/images/catalogo.jpg
2. Recarregue a página (Ctrl+F5)
3. Pronto! ✅
```

### Opção 2: Sua imagem tem outro nome
```
1. Edite index.html (linha ~135)
2. Procure por: src="images/catalogo.jpg"
3. Substitua "catalogo.jpg" pelo nome da sua imagem
4. Salve e recarregue (Ctrl+F5)
```
